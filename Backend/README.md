# Back-end 

### Exemplo do .env
Criar o arquivo .env na raiz e configurar as variáveis de ambiente.
```sh
# Exemplo
HOST = 127.0.0.1
DATABASE = tcc
DATABASE_PORT = 3306
DATABASE_USER = root
DATABASE_PASSWORD = root
SECRET = 0aa1ea9a5a04b78d4
```

### Executando o projeto Back-end

```sh
$ npm install
$ npm run dev
```
